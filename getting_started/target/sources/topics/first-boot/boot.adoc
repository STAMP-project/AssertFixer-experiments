
=== Booting the Server


:include_filename: getting_started/topics/first-boot/boot.adoc
include::../templates/header.adoc[]


To boot the {project_name} server, go to the `bin` directory of the server distribution and run the `standalone` boot script:

.Linux/Unix
[source]
----
$ .../bin/standalone.sh
----

.Windows
[source]
----
> ...\bin\standalone.bat
----
